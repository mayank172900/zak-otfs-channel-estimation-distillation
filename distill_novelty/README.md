# Distill Novelty

`distill_novelty/` implements a fast teacher-student compression path for the frozen Zak-OTFS baseline CNN.

The main idea is simple:

- teacher: the frozen baseline CNN from `logs/checkpoints/full_cnn_best.pt`
- student: a much smaller single-branch CNN
- input: `H_obs` from the Phase 1 memmap dataset
- training targets:
  - primary: teacher estimate `H_base`
  - secondary: true support channel `H_true`

This keeps the paper scope safe:

- preserve the strong baseline behavior as much as possible
- reduce parameter count and inference cost
- measure the accuracy-efficiency tradeoff instead of trying to replace the baseline with a riskier method

## Folder layout

- `configs/`: training, smoke, and evaluation configs
- `scripts/`: train, evaluate, and benchmark entry points
- `src/zakotfs_distill/`: dataset, model, training, evaluation, and benchmark code
- `tests/`: smoke tests for the full path

## Model

The student uses the same real/imag split pattern as the teacher:

- run one small real-valued CNN on the real support image
- run the same CNN on the imaginary support image
- combine both outputs into a complex support estimate

Provided variants:

- `lite_l`: about `40k` params
- `lite_m`: about `23k` params
- `lite_s`: about `6k` params

The teacher has about `245k` params, so even `lite_l` is much smaller.

## Distillation loss

The student prediction is trained with:

- teacher loss: normalized error to `H_base`
- truth loss: normalized error to `H_true`

Total loss:

- `L = distill_weight * L_teacher + truth_weight * L_truth`

Recommended default:

- `distill_weight = 0.8`
- `truth_weight = 0.2`

## Data dependency

This pipeline reuses the memmap datasets generated by `physics_novelty/Phase 1`.

Required manifests for full training:

- `physics_novelty/results/phase1_datasets/train_phase1.json`
- `physics_novelty/results/phase1_datasets/val_phase1.json`

Smoke mode uses:

- `physics_novelty/results/phase1_datasets/smoke_phase1.json`

## Commands

Smoke training:

```bash
python distill_novelty/scripts/train_student.py --config distill_novelty/configs/distill_smoke.yaml
```

Alternative student variants:

```bash
python distill_novelty/scripts/train_student.py --config distill_novelty/configs/distill_lite_m.yaml
python distill_novelty/scripts/train_student.py --config distill_novelty/configs/distill_lite_s.yaml
```

Smoke evaluation:

```bash
python distill_novelty/scripts/run_distill_eval.py --config distill_novelty/configs/distill_smoke.yaml
```

Smoke benchmark:

```bash
python distill_novelty/scripts/benchmark_student.py --config distill_novelty/configs/distill_smoke.yaml
```

Full training:

```bash
python distill_novelty/scripts/train_student.py --config distill_novelty/configs/distill_train.yaml
```

Full evaluation:

```bash
python distill_novelty/scripts/run_distill_eval.py --config distill_novelty/configs/distill_eval_full.yaml
```

Full benchmark:

```bash
python distill_novelty/scripts/benchmark_student.py --config distill_novelty/configs/distill_eval_full.yaml
```

## Outputs

Training writes to `distill_novelty/logs/`.

Evaluation writes to `distill_novelty/results/`:

- `distill_nmse_vs_pdr_<variant>_<mode>.csv|json|png`
- `distill_nmse_vs_snr_<variant>_<mode>.csv|json|png`
- `distill_ber_vs_pdr_<variant>_<mode>.csv|json|png`
- `distill_ber_vs_snr_<variant>_<mode>.csv|json|png`

Benchmark writes:

- `distill_benchmark_<variant>_<mode>.json`
